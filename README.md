# 공교육 특화 AX 전략 핸드북

공교육의 인공지능전환(AX, AI Transformation)을 왜·무엇을·어떻게 추진할지 정리한 교사·학교 관리자·교육정책 담당자용 안내서입니다. 정책·전략·현장·윤리를 하나로 엮었으며, 2026년 상반기 한국 공교육 정책을 기준으로 공식 자료에 근거해 작성했습니다.

## 구성
- **1장 AX 개관과 공교육의 특수성** — 전산화·DX·AX의 차이, 공교육이 기업과 다른 다섯 가지 조건
- **2장 국내외 정책·동향** — 한국 정책의 골격과 AI 디지털교과서의 부침, UNESCO·OECD와 주요 7개국 전략
- **3장 6대 영역 전략 프레임워크** — 거버넌스·인프라·교원 역량·교육과정·평가·윤리 + 단계별 로드맵
- **4장 학교 현장 적용** — 수업·행정 활용 매뉴얼과 초·중·고급 실습 시나리오
- **5장 윤리·안전·개인정보** — 교육분야 AI 윤리원칙, 생성형 AI의 위험, 개인정보·연령 규칙, 도구 도입 체크리스트
- **6장 점검 문항 / Q&A** — 연수·자가 학습용 객관식·단답·서술형·개념 Q&A

## 대상·기준
- 대상: 한국 초·중·고 공교육 교사, 학교 관리자, 교육정책 담당자
- 기준: 2026년 상반기 정책, 자료 수집일 2026-07-05
- 주요 출처: 교육부, KERIS, 개인정보보호위원회, AI 기본법, UNESCO, OECD 등

정책은 계속 변화하므로 각 장에 표시된 출처에서 최신 상태를 확인하며 읽기를 권합니다.
